Qu'est-ce que Foodie ?

Foodie est une solution de commande de nourriture en ligne permettant aux utilisateurs de commander facilement. Elle comprend 3 applications : une application restaurant pour recevoir les commandes, une application livreur pour gérer les livraisons, ainsi qu'un panneau web pour l'administration et les restaurants.
